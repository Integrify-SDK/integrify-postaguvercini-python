# PostaGuvercini mühit dəyişənləri

## Haqqında

Aşağıdakı cədvəldə bütün mühit dəyişənləri ilə tanış ola bilərsiniz.

| Dəyişənin adı              | Məcburi?                  | Mənası                    | Default dəyəri |
| -------------------------- | ------------------------- | ------------------------- | -------------- |
| `POSTA_GUVERCINI_USERNAME` | :fontawesome-solid-check: | PostaGuvercini username-i | `-`            |
| `POSTA_GUVERCINI_PASSWORD` | :fontawesome-solid-check: | PostaGuvercini password-u | `-`            |
